화면에서 hide된 상태에서는 이벤트가 발생하지 않는다.
window load, resize 시에 포지션을 자동으로 업데이트 한다.
						
							/**

							 * event
							 */
							$( '.target' ).on( 'bonding-rect-activate bonding-rect-deactivate bonding-rect-between', function (e) {
								console.log( e.type, e.progress );
							});

							/**
							 * 조건에 맞는지 체크하여 이벤트 발생
