회원님은 아래와 같은 이유로 탈퇴가 불가능 합니다.

회원탈퇴 불가 내역으로 상세 내용 및 탈퇴 방법으로 구성
상세내용 탈퇴 방법
진행중인 거래내역 : 2 건 거래 종료 및 취소
환불 관련 : 0 건 환불처리 종료
인출가능한 포인트 및 캐시(예치금) : 0 건 예치금 사용
회원탈퇴 시 유의사항
  • 탈퇴 시 보유하고 계신 쿠폰과 모든 포인트는 즉시 자동 소멸됩니다.
  • 탈퇴 시 아모레퍼시픽 통합멤버십도 동시 탈퇴 됩니다.
  • 진행 중인 전자상거래 이용내역(배송/교환/반품 중인 상태)에 있거나 이용하신 서비스가 완료되지 않은 경우 탈퇴하실 수 없습니다.
  • 회원탈퇴하셔도 바로 재가입 가능하며 동일 아이디는 사용할 수 없습니다.
회원정보 보존 안내
  • 회원탈퇴와 함께 고객님이 회원가입 시 입력하신 모든 개인정보는 바로 삭제됩니다.
  • 온라인 회원탈퇴 후에도 오프라인 가입정보가 있을 경우, 포인트는 소멸되지 않습니다.
